# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - March 19, 2026

### Added
- 16 specialized Salesforce agents (Apex Architect, LWC Builder, Test Writer, etc.)
- Multi-model AI support (Claude, Gemini, GPT-4)
- Governor limit validation and enforcement
- FSD (Feature-Sliced Design) architecture support
- Token tracking and cost management
- Context-aware project scanning
- Comprehensive CLI interface (`sfai` command)
- Memory system for persistent context
- SOQL query optimization
- Security scanning and guardrails
- Complete documentation and guides
- Example projects and templates

### Features
- **16 Agents:** Specialized agents for different Salesforce domains
- **Multi-Model:** Support for Claude 3, Gemini, and GPT-4
- **Guardrails:** Automatic enforcement of Salesforce best practices
- **Token Tracking:** Real-time cost monitoring across models
- **Context Engine:** Deep project analysis and pattern recognition
- **CLI-First:** Powerful terminal interface for developers
- **Memory System:** Persistent shared memory between agents
- **FSD Support:** Feature-Sliced Design pattern enforcement

### Documentation
- Installation guide with troubleshooting
- Quick start guide for first-time users
- Comprehensive CLI usage documentation
- Architecture overview and design patterns
- Agent reference documentation (all 16 agents)
- API reference for core modules
- Concepts guide (context awareness, orchestration, etc.)
- Deployment and hosting guides
- Known limitations and comparison with other tools

### Testing
- Unit tests for core modules
- Integration tests for agent workflows
- Token tracking accuracy tests
