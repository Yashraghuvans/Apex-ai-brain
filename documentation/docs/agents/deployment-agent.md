# Additional Agents

Quick reference for remaining specialized agents.

---

## Deployment Agent

Handles deployment planning and release management.

```bash
sfai /spawn deployment-agent "Plan deployment for lead scoring feature"
```

---

## Diff Reviewer

Analyzes code changes and git differences.

```bash
sfai /spawn diff-reviewer "Analyze changes between main and feature branch"
```

---

## Flow Advisor

Designs Salesforce Flows and automation workflows.

```bash
sfai /spawnflow-advisor "Design approval flow for contract requests"
```

---

## Git Agent

Manages git operations, commits, and releases.

```bash
sfai /spawn git-agent "Create commit message for lead scoring"
```

---

## Kanban Agent

Manages project planning and sprint organization.

```bash
sfai /spawn kanban-agent "Create sprint backlog for portal feature"
```

---

## LWC Builder

Creates Lightning Web Components.

```bash
sfai /spawn lwc-builder "Build a filterable data table component"
```

---

## Memory Agent

Manages knowledge base and shared memory.

```bash
sfai /spawn memory-agent "Retrieve all previous Lead designs"
```

---

## Metadata Manager

Configures org metadata and settings.

```bash
sfai /spawn metadata-manager "Setup custom settings for feature flags"
```

---

## Schema Analyst

Designs data models and custom objects.

```bash
sfai /spawn schema-analyst "Design portal data model"
```

---

## Security Agent

Conducts security audits and vulnerability checks.

```bash
sfai /spawn security-agent "Audit authentication handler for vulnerabilities"
```

---

## SOQL Optimizer

Optimizes SOQL queries for performance.

```bash
sfai /spawn soql-optimizer "Optimize dashboard queries"
```

---

## Test Writer

Generates comprehensive test suites.

```bash
sfai /spawn test-writer "Write comprehensive tests for LeadService"
```

---
