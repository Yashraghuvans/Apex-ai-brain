# Kanban Agent

Project management and sprint planning expert.

## Overview

Manages task breakdowns, sprint planning, and agile workflows.

## Use Cases

 Create sprint backlogs
 Break down epic features
 Plan sprints
 Track progress

## Command

```bash
sfai /spawn kanban-agent "Create sprint backlog for portal feature"
```
