# Security Agent

Security and compliance audit specialist.

## Overview

Conducts security audits, identifies vulnerabilities, and ensures compliance.

## Use Cases

 Audit security
 Find vulnerabilities  
 Check compliance
 Recommend hardening

## Command

```bash
sfai /spawn security-agent "Audit authentication handler for vulnerabilities"
```
