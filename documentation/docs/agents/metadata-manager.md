# Metadata Manager Agent

Salesforce metadata configuration expert.

## Overview

Configures and manages Salesforce metadata, custom settings, and org configuration.

## Use Cases

 Configure custom settings
 Manage metadata
 Org configuration
 Feature flag setup

## Command

```bash
sfai /spawn metadata-manager "Setup custom settings for feature flags"
```
