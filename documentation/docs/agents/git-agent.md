# Git Agent

Version control and release management specialist.

## Overview

Handles git workflows, commits, branches, and release management.

## Use Cases

 Generate commit messages
 Create PR descriptions
 Manage branches
 Generate CHANGELOG

## Command

```bash
sfai /spawn git-agent "Create meaningful commit for lead scoring"
```
