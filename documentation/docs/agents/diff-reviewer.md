# Diff Reviewer Agent

Analyzes code changes and pull requests.

## Overview

Reviews code commits and git diffs, providing change analysis and improvement suggestions.

## Use Cases

 Analyze code changes
 Review pull request diffs
 Generate change summaries
 Suggest improvements

## Command

```bash
sfai /spawn diff-reviewer "Review changes between main and feature branch"
```
