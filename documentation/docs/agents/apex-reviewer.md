# Apex Reviewer Agent

Code quality, pattern validation, and best practices review.

## Overview

The Apex Reviewer performs comprehensive code reviews focusing on maintainability, patterns, and enterprise standards.

## Use Cases

✅ Review Apex classes for best practices
✅ Validate design pattern usage
✅ Check code quality and complexity
✅ Identify refactoring opportunities

## Command

```bash
sfai /spawn apex-reviewer "Review <class-name> for best practices"
```
