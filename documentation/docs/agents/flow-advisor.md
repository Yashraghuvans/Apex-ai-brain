# Flow Advisor Agent

Salesforce Flow design and automation expert.

## Overview

Designs Salesforce Flows, automations, and process workflows.

## Use Cases

 Design approval workflows
 Create process automations
 Optimize flow logic
 Handle decision trees

## Command

```bash
sfai /spawn flow-advisor "Design multi-step approval flow"
```
