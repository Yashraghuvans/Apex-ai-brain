# Memory Agent

Knowledge base and memory management specialist.

## Overview

Manages shared memory, retrieves context, and maintains the knowledge base.

## Use Cases

✅ Retrieve previous outputs
✅ Manage memory entries
✅ Search knowledge base
✅ Organize context

## Command

```bash
sfai /spawn memory-agent "Find all previous Account designs"
```
